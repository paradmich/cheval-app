# Cheval Holdings — Private Wealth OS

A private, AI-powered **family-office operating system** for Cheval Holdings — one consolidated place to monitor every account, holding, revenue line, fund, and governance function, with a read-only / watch-only security model.

## Security principle

The app **reads and reports — it never moves funds.** Every connection is read-only:
Gemini (Auditor key), Fidelity IRA (read-only), Charles Schwab (read-only), TradeSmart (read endpoints), watch-only crypto wallets (xpub), and bank aggregation. Sensitive data is field-level encrypted with 2FA, RLS, and an audit log.
